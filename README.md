# Химчистка

Тема проекта: **химчистка**.

Этот репозиторий теперь содержит стартовое C#-решение для **Visual Studio**:
- `Himchistka.sln` — solution-файл;
- `src/Himchistka.App` — консольное приложение с примером услуг.

## Как открыть в Visual Studio

1. Откройте Visual Studio 2022.
2. Нажмите **Open a project or solution**.
3. Выберите `Himchistka.sln`.
4. Запустите проект кнопкой **Start** или клавишей `F5`.

## Что уже есть в проекте

- Модель `Service` (услуга химчистки).
- Небольшой список услуг с ценами и сроками.
- Вывод информации в консоль.
